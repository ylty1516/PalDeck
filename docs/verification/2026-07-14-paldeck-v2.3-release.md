## 更新与安全边界

- 更新 API：`https://api.github.com/repos/ylty1516/PalDeck/releases/latest`。
- Release 下载只接受 `github.com/ylty1516/PalDeck/releases/download/...`。
- 同名 `.sha256` sidecar 和 SHA-256 校验仍为强制要求。
- 旧仓库不作为 fallback。
- Nexus 继续匿名只读且成人内容 fail-closed。
- Workshop 继续只管理已订阅项目启停，不订阅、退订或删除。

## 残余风险

- 仅支持 Windows 10/11 Steam 客户端；不支持 Xbox/Microsoft Store 和专用服务器。
- 外部或未知 UE4SS 用户文件会被保留而非删除；无所有权安装只操作固定内置包已知路径。
- UE4SS 可变配置不参与内容摘要异常判定，缺失、目录替代和 reparse 冲突仍会 fail-closed。
- 损坏回收记录只显示诊断，不执行负载恢复或清理，需要用户保留数据后人工处理。
