# PalDeck v2.3 自动更新仓库迁移实现计划

> **面向 AI 代理的工作者：** 必需子技能：使用 superpowers:subagent-driven-development（推荐）或 superpowers:executing-plans 逐任务实现此计划。步骤使用复选框（`- [ ]`）语法来跟踪进度。

**目标：** 将应用内更新唯一可信源从旧仓库迁移到 `ylty1516/PalDeck`，继续强制 GitHub Release URL、同名 sidecar 和 SHA-256 校验。

**架构：** 保留现有 `self_updater` 下载与替换流程，只替换信任根并收紧测试。旧仓库不作为 fallback，避免攻击面和版本分叉。

**技术栈：** Python 3.13、urllib、GitHub Releases、pytest。
