- [ ] **步骤 3：实现模型**

```python
@dataclass(frozen=True)
class OwnedFrameworkFile:
    relative_path: str
    size: int
    sha256: str
    mutable: bool

@dataclass(frozen=True)
class OwnershipRecord:
    schema_version: int
    game_fingerprint: str
    source: Literal["bundled", "upstream", "local_zip"]
    asset_name: str
    asset_sha256: str
    installed_at: str
    files: tuple[OwnedFrameworkFile, ...]
    framework_mods: tuple[str, ...]

@dataclass(frozen=True)
class OwnershipAudit:
    integrity: Literal["healthy", "missing", "modified", "conflict"]
    missing: tuple[str, ...]
    modified: tuple[str, ...]
    conflicts: tuple[str, ...]
```

`OwnershipStore` 使用单个 schema v1 JSON 按 game fingerprint 映射；路径验证与 manifest/trash 同级严格；audit 只访问 `Win64 / relative_path`。非 mutable 文件比较大小和摘要；mutable 文件只检查存在性与路径类型，用户内容变化不降低完整性。

`install_from_zip/install_from_bytes/_install_archive` 增加 keyword-only `preserve_mutable=False`。

- preserve 且 settings 已存在：不替换，事务内只补丁 `bUseUObjectArrayCache`，其他字节语义保持。

- preserve 且 mods.json 已存在：不写。
