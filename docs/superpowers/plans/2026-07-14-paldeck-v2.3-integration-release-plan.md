# PalDeck v2.3 安全生命周期集成与发布计划

> **面向 AI 代理的工作者：** 必需子技能：使用 superpowers:subagent-driven-development（推荐）或 superpowers:executing-plans 逐任务实现此计划。步骤使用复选框（`- [ ]`）语法来跟踪进度。

**目标：** 将回收站、外部 Mod、PalDeck 更新仓库和 UE4SS 生命周期接入现有 Flask/WebView 界面，完成 v2.3.0 视觉、Frozen smoke、可复现构建和正式发布。

**架构：** Flask 暴露固定字段 API，前端通过现有 `data-action` 委托调用；“我的模组”提供外部标签和回收站入口，设置页提供忽略重置和 UE4SS 生命周期。保留五页导航，不新增独立页面。

**技术栈：** Flask、原生 ES Modules、CSS Grid/Flex、Edge/WebView2 截图、PyInstaller、PowerShell、pytest、Node test。

**前置依赖：** 按顺序完成 trash-external、updater-migration、ue4ss-lifecycle 三份计划。
