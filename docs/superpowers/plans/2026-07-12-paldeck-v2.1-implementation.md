# PalDeck v2.1 实现计划

> **面向 AI 代理的工作者：** 必需子技能：使用 superpowers:subagent-driven-development（推荐）或 superpowers:executing-plans 逐任务实现此计划。步骤使用复选框（`- [ ]`）语法来跟踪进度。

**目标：** 发布 PalDeck v2.1.0，提供三套重制樱花、内置 Palworld 专用 UE4SS、Steam Workshop 精确检测/真实启停和独立开源致谢页。

**架构：** `ue4ss_provider` 将第三方资源信任与既有事务安装器分离；`steam_workshop` 只扫描 Palworld 固定 Workshop 路径并原子维护 `PalModSettings.ini`；纯函数式 `petal-engine.js` 与 Canvas renderer 分离；`credits` 提供固定目录和可信外链 allowlist。现有 Flask cookie 鉴权、ModService 和单 DOM 前端继续复用。

**技术栈：** Python 3.13、Flask、pytest、原生 JavaScript ES modules、Canvas 2D、Node 22 内置 `node:test`、PyInstaller、PowerShell、GitHub CLI。
