# PalDeck v2.3 回收站与外部 Mod 实现计划

> **面向 AI 代理的工作者：** 必需子技能：使用 superpowers:subagent-driven-development（推荐）或 superpowers:executing-plans 逐任务实现此计划。步骤使用复选框（`- [ ]`）语法来跟踪进度。

**目标：** 将本地 Mod 永久删除替换为可恢复 30 天的安全回收站，并让首次扫描发现的外部 Mod 可持久取消管理和重新发现。

**架构：** `ue4ss_config.py` 提供无文件路径副作用的配置合并；`trash_store.py` 只负责严格记录；`trash_service.py` 负责同卷负载与本地 Mod 生命周期；`ignored_mod_store.py` 负责规范化忽略身份。`ModService` 只编排活动清单，所有危险路径都由可信根和枚举派生。

**技术栈：** Python 3.13、Flask、dataclasses、pathlib、现有 `JsonStore`/`ManifestStore`/`game_write_lock`、pytest。
